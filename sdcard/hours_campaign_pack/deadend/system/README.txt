Dead End Inc. long campaign pack

This pack is meant to stretch the current firmware into a much longer run without changing the code.
It uses the existing SD loader and fills the live systems with far more usable content.

What this pack adds:
- 24 products total
- 24 districts total
- 48 contacts total
- 120 events total
- 16 campaign stages

What to copy:
Copy the whole deadend folder from this pack to the root of the SD card.

Important:
- Keep your existing /deadend/saves/ folder if you want to preserve a run.
- Replace the content folders if you want the new campaign and long-form progression.

Why it lasts longer:
- far more map paging and contact rotation
- longer objective ladder
- more event variety across later days
- more product focus changes and district demand spread

Auto-loaded folders used by this pack:
/mods/products/
/mods/districts/
/mods/characters/
/mods/events/
/mods/campaign/

This pack does not require a firmware reflash.

Product format also supports:
id|label|basePrice|risk|cookCost|batchYield|heatGain|requiredLab|requiredRep|desc

Older product files still load. If requiredLab/requiredRep are missing, the firmware derives defaults.
